1. Create `packages/shell/src/stores/useWindowStore.ts` implementing the following store interface:

```typescript
import { create } from 'zustand'
import type { WindowDescriptor, WindowRect, WindowState } from '@vidorra/types'

interface WindowStoreWindow extends WindowDescriptor {
  preMaximizeRect?: WindowRect
  minWidth: number
  minHeight: number
}

interface WindowStore {
  windows: WindowStoreWindow[]
  nextZIndex: number

  openWindow: (descriptor: Omit<WindowDescriptor, 'zIndex' | 'focused'> & {
    minWidth?: number
    minHeight?: number
  }) => void
  closeWindow: (id: string) => void
  focusWindow: (id: string) => void
  setWindowState: (id: string, state: WindowState) => void
  setWindowRect: (id: string, rect: WindowRect) => void
  toggleMaximize: (id: string) => void
}
```

Key implementation details:
- `openWindow`: Staircase offset — count non-minimized windows, multiply by 20, add to base `rect.x` and `rect.y`. Assign `nextZIndex` to new window. Set `focused: true` on new window, `focused: false` on all existing. Default `minWidth: 200`, `minHeight: 150`.
- `closeWindow`: Filter out window by id. If the closed window was focused, focus the topmost remaining window (highest zIndex).
- `focusWindow`: Set target `focused: true`, `zIndex: nextZIndex`, increment nextZIndex. Set all others `focused: false`.
- `setWindowState('minimized')`: Also set `focused: false`. Focus next highest zIndex window.
- `setWindowState('maximized')`: Save `preMaximizeRect` from current `rect` before changing state.
- `setWindowState('normal')`: If `preMaximizeRect` exists, restore `rect` from it and clear `preMaximizeRect`.
- `toggleMaximize`: If `state === 'maximized'`, call `setWindowState(id, 'normal')`; else call `setWindowState(id, 'maximized')`.
- `setWindowRect`: Update `rect` for the given window id.
