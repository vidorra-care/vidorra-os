2. Create `packages/shell/src/components/Dock/DockItem.tsx`:
   - Props: `{ app: AppManifest; mouseX: MotionValue<number>; isRunning: boolean; onOpen: (app: AppManifest) => void }`
   - Use `useRef<HTMLDivElement>(null)` to get icon bounding rect.
   - Derive distance from `mouseX` using `useTransform`:
     ```typescript
     const distance = useTransform(mouseX, (val: number) => {
       const bounds = ref.current?.getBoundingClientRect()
       if (!bounds) return Infinity
       return val - bounds.x - bounds.width / 2
     })
     ```
   - Multi-point interpolation for width using `useTransform`:
     ```typescript
     const BASE_SIZE = 48
     const MAX_SIZE = 80
     const DISTANCE_LIMIT = BASE_SIZE * 6 // 288px
     const widthTransform = useTransform(
       distance,
       [-DISTANCE_LIMIT, -DISTANCE_LIMIT/1.25, -DISTANCE_LIMIT/2, 0, DISTANCE_LIMIT/2, DISTANCE_LIMIT/1.25, DISTANCE_LIMIT],
       [BASE_SIZE, BASE_SIZE*1.1, BASE_SIZE*1.414, MAX_SIZE, BASE_SIZE*1.414, BASE_SIZE*1.1, BASE_SIZE]
     )
     ```
   - Smooth with `useSpring(widthTransform, { stiffness: 150, damping: 15, mass: 0.1 })`.
   - Render `<motion.img src={app.icon} alt={app.name} style={{ width, height: width }} draggable={false} className={styles.dockIcon} />`.
   - Running indicator: `<span className={`${styles.dot} ${!isRunning ? styles.dotHidden : ''}`} data-testid="running-dot" />`. The `data-testid` attribute enables test queries.
   - Hover state: Track with `useState<boolean>(false)` on `onMouseEnter`/`onMouseLeave`. Show tooltip on hover.
   - Launch bounce animation: Use `useAnimation()` from Framer Motion. On click (when app is not running), trigger: `controls.start({ y: [0, -10, 0], transition: { duration: 0.4, ease: 'easeInOut' } })`. Then call `onOpen(app)`.
   - Right-click: `onContextMenu` handler that prevents default and opens the shared ContextMenu via local state `{ x, y }`. Items depend on `isRunning`:
     - Not running: `[{ label: '打开', action: () => onOpen(app) }]`
     - Running: `[{ label: '打开', action: () => focusExistingWindow(app.id) }, { label: '在 Dock 中隐藏', action: () => {} }, { label: '关闭', action: () => closeAllWindowsForApp(app.id) }]`
   - `closeAllWindowsForApp`: Gets windows from store where `appId === app.id`, calls `closeWindow` for each.
