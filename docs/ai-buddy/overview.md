# AI Buddy — 概览

## 定位

AI Buddy 是 Vidorra OS 的系统内置 AI 助手，类似 Claude Code 在终端中的体验，但运行在 Web 桌面环境中。

它不是一个普通的 App，而是系统的一部分：
- 能感知当前桌面状态（哪些 App 在运行、聚焦的是什么）
- 能直接操作系统（打开 App、读写 DataStore、执行 Spotlight 搜索）
- 所有操作经过完整的权限系统，不绕过安全边界
- 支持流式输出，响应实时呈现

---

## 核心体验目标

参考 Claude Code 和 OpenClaw 的交互模式：

1. **自然语言操作系统**：「帮我打开记账本，新建一条今天的午饭记录」
2. **上下文感知**：AI 知道你当前在用什么 App，不需要重复说明
3. **工具调用透明**：AI 执行操作前展示将要做什么，用户确认后执行
4. **流式响应**：回答实时流出，不等待完整响应
5. **多模态输入**：支持截图、文件拖入（P3）

---

## 模块组成

```
ai-buddy/
├── BuddyCore          # 与 AI 模型通信，管理对话上下文
├── BuddyShell         # 浮动面板 UI
├── SystemContext      # 向 AI 注入系统上下文
├── ToolExecutor       # AI 可调用的系统工具
└── PermissionBridge   # 工具调用的权限校验
```

## UI 形态

```
┌─────────────────────────────┐
│  ✦ Buddy              — □ × │  ← 标题栏，可拖拽
├─────────────────────────────┤
│                             │
│  你好！我可以帮你操作系统、  │
│  查询数据、打开应用。        │
│                             │
│  ─────────────────────────  │
│                             │
│  > 帮我查一下本月支出总额    │
│                             │
│  正在查询记账本数据...       │
│  ┌─────────────────────┐    │
│  │ 工具调用             │    │
│  │ data.query          │    │
│  │ com.budget:tx       │    │
│  └─────────────────────┘    │
│                             │
│  本月（6月）总支出：¥3,240  │
│  其中餐饮 ¥890，交通 ¥320   │
│                             │
├─────────────────────────────┤
│  [输入框]              [发送] │
└─────────────────────────────┘
```

### 最小化状态

收起为 Dock 中的一个图标，有未读消息时显示角标。

### 全屏模式

类似 Claude Code 的全屏终端体验，适合复杂任务。
